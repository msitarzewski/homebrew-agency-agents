# Homebrew Agency Agents

Homebrew tap for [Agency Agents](https://agencyagents.app) — a native installer for AI agents.

## Install

```sh
brew tap msitarzewski/agency-agents
brew install --cask agency-agents
```

Installs the signed + notarized `.dmg` (Apple Silicon or Intel, auto-selected). macOS 13+.

Upgrade with `brew upgrade --cask agency-agents`.
